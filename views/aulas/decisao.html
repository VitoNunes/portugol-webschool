Estrutura de Decisão
As estruturas condicionais possibilitam a execução de um conjunto de comandos quando determinada condição for satifeita ou não. 
    se(frango_desfiado == "possui" e farinha == "possui"){                
       escreva("É possível fazer a torta")                  
    }senao{                  
       escreva("Não é possível fazer a torta")            
    }

Estrutura Se e Senao
A estrutura “se” e “senao” trabalham juntas, ou seja, uma depende da outra. Quando estabelecemos uma condição a qual irá ser verificada caso a condição seja satisfeita serão executados um conjunto de comandos (os quais pertence a estrutura “se”). Se a condição não for satisfeita irá ser executados os comandos pertencentes ao bloco do “senao” 
    se(condição) {
      comandos
    }
    senao {
      comandos
    }

Se e senao encadeado
O se e senao encadeado consiste na ideia de ir verificando várias condições sucessivamente. 
    se(condição){
       comandos                
    }senao se(condição){                  
       comandos           
    }senao se(acondição){                  
       comandos           
    }senao{                  
       comandos          
    }

Se e Senao aninhado
Já o se e senao aninhado trata-se da ideia de verificar uma condição caso outra condição já tenha sido verificada. 
    se(condição){
       se(condição){
         comandos
       }senao se(condição){                  
         comandos           
       }
    }senao se(condição){                  
       se(condição){
         comandos
      }senao se(condição){                  
        comandos           
      }           
   }

Estrutura Escolha Caso
Esta estrutura possui uma funcionalidade muito parecida com o se e senao encadeado, porém essa só pode ser aplicada em variáveis do tipo inteiro e caracter. 
    Escolha (variável_de_verificação){

      caso valor1:
         comandos
         pare //impede a execução do próximo caso
         
      caso valor2:
         comandos
         pare //impede a execução do próximo caso
         
     caso valor3:
        comandos
        pare //impede a execução do próximo caso
        
     }

Estrutura Escolha Caso Contrário
A estrutura caso contrário tem a mesma funcionalidade que a estrutura anterior, o diferencial é que agora temos uma estrutura (caso contrário) que é executada quando as condições propostas não são satisfeitas. 
    Escolha (variável_de_verificação){

      caso valor1:
         comandos
         pare //impede a execução do próximo caso
         
      caso valor2:
         comandos
        pare //impede a execução do próximo caso
         
     caso contrario: //é executado para qualquer opção diferente dos casos anteriores
        comandos
    
    }
