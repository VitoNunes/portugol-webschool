Vetor
O vetor a grosso modo é uma variável que armazena variáveis do mesmo tipo. Nele as variáveis recebem o nome de elemento e são identificadas pelo índice. Podemos declarar vetores de duas formas:

A primeira é especificando quantas posições ele terá, fazendo com que o usuário possa escolher os valores que irão preencher o vetor (tipo_de_dado nome_do_vetor [quantidade_de_elementos]). 
1    inteiro vetor[5]
2    
3    //Preenchendo o vetor
4    para(inteiro i = 0; i <= 4; i++){                
5       escreva("Digite um número: \n")
6       leia(vetor[i]) //Aqui definimos o indice do vetor como i, para que o índice o acompanhe
7    }
6    //Exibindo os números
7    escreva("Os números digitados foram: \n\n")
8    
9    para(inteiro c = 0; c <= 4; c++){                
10      escreva(vetor[c], " ")
11   }

A segunda é preenchendo o vetor préviamente com os valores que irão compr ele (tipo_de_dado nome_do_vetor []={elementos}). 
1    cadeia frutas[] = {"banana", "maça", "laranja", "pera", "uva"}//Anotação 1
2     
3    escreva("As frutas são: ")
4    para(inteiro i = 0; i < 5; i++){    
5       
6       //O if abaixo é usado para que no indice 4 a ", " não seja repetida
7       se(i != 4){
8         escreva(frutas[i], ", ")
9       }senao{
10        escreva(frutas[i])
11      }
12   }

Matrizes
As matrizes assim como os vetores são variáveis que armazenam dados do mesmo tipo. O diferencial consiste que a matriz é um vetor bidimensional. Um bom exemplo para se guiar é a tabela do jogo Sudoku, nela temos uma malha quadriculada de 3x3 que se repete 9 vezes, a matriz armazena os valores de forma parecida.

Assim como no vetor, podemos declarar matrizes de duas formas: A primeira permite que o usuário insira dados, com indíces para linha e coluna pré determinados (tipo_de_dado nome_da_matriz[quantidade_de_linhas][quantidade_de_colunas]). 
1    cadeia matriz[3][3]
2    //Preenche com os valores digitados pelo usuário
3    para(inteiro l = 0; l <= 2; l++){ //Definirá o indíce das linhas
4       para(inteiro c = 0; c <= 2; c++){ //Definirá o indíce das colunas
5	     
6	         escreva("digite o nome de uma fruta: ")
7 	         leia(matriz[l][c]) //Acompanha os dois índices
8 	         escreva("\n")
9 	 }
10 	 }
11	
12	 escreva("Frutas:\n")
13   //Mostra os valores em forma de tabela
14	 para(inteiro l = 0; l <= 2; l++){
15	     para(inteiro c = 0; c <= 2; c++){
16	    
17	         escreva(matriz[l][c], " ")
18	         se(c == 2){
19	             escreva("\n")
20	         }
21	        
22	 }
23	 }
